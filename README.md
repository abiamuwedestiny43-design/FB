# Facebook Connect Integration

A premium registration system that allows users to register using their Facebook details.

## Features
- **Modern Landing Page**: High-end aesthetics with glassmorphism and animated backgrounds.
- **Facebook Login Flow**: Realistic modal for capturing registration details.
- **Admin Dashboard**: Secure-looking interface for administrators to monitor and view registered users' login details.
- **Data Persistence**: Uses `localStorage` to keep data across sessions.

## How to Run
1. Ensure dependencies are installed:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open the provided URL (usually `http://localhost:5173`).

## Usage
- **User Flow**: Click "Continue with Facebook" -> Enter credentials -> Success notification.
- **Admin Flow**: Click the **Dashboard** icon in the top right corner to view captured credentials.
